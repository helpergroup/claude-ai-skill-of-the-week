## ⚡ How to use a Skill

1. Click on the skill file (SKILL.md)
2. Click the **Raw** button (top right)
3. Save the file as `.md` on your computer
4. In Claude.ai → **Settings** → **Custom Instructions** → upload the file
   *(or drag it directly into the chat)*

## 🙌 Submit a Skill

Found a Claude Skill that genuinely changed how you work?

We're always looking for the next hidden gem — and we want to hear from you.

If you've tested a skill that deserves a spot in the newsletter,
submit it here. Every skill we publish gets full credit to the person
who sent it.

**What makes a good submission:**

- Solves a real, specific problem
- Works without technical setup (no code, no Claude Code)
- You've actually made it / used it — not just found it in a list

**How to submit:**

Send your skill to **aiskilloftheweek@gmail.com** with:

- The SKILL.md file in attachment
- 2-3 lines on what it does and when you use it

If we publish it in the newsletter → you get full credit in the issue + a shoutout on LinkedIn!

> or that you have the right to share it.
